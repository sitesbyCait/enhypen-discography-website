# ENHYPEN Discography Website
A fan-made interactive archive designed to help ENGENEs and new fans to explore ENHYPEN's discography, members, milestones, and era-based content in one organized website.

## Overview
This project focuses on creating a visually immersive and easy-to-navigate fan experience. Instead of browsing content across different platforms (Youtube, Weverse, X), users can explore ENHYPEN's music, member profiles, eras, videos, and fan-centered content through one cohesive digital archive.

## Tech Stack
- React
- Tailwind CSS
- JavaScript
- Git / GitHub
- Figma

## Key Features
- Cinematic hero section inspired by ENHYPEN's debut era.
- Member cards with hover action and dedicated profile pages.
- Discography section with albums, tracklist, and media.
- List of Awards
- "For Engenes" section featuring photos and meaningful captions from the members.
- Responsive layout for desktop and mobile
