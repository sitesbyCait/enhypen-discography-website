# Project Brief

## Project Name
ENHYPEN Discography Website

## Project Type
Fanmade interactive archive / portfolio project

## Purpose
The purpose of this project is for ENGENEs and new fans to have easier access to organized and immersive content of ENHYPEN's discography, members, milestones, and media content.

## Problem
ENHYPEN content is spread across different platforms such as Youtube, Weverse, X, and fan accounts. For new fans, it can feel overwhelming to know where to start.

## Solution
This website brings key ENHYPEN content into one structured experience. It combines discography, members, awards, and curated media in an engaging way.

## Target Users
- New ENGENEs who want to learn more about ENHYPEN
- Existing ENGENEs who want to revisit eras
- Kpop fans interested in ENHYPEN's music and journey
- Viewers who enjoy interactive fanmade websites

## Project Goals
- Make ENHYPEN content easier to browse
- Create an aesthetic and emotional fan experience
- Practice UI/UX design and front-end development
- Build a portfolio-worthy web project
